# 📊 Calculadora de Média em C

> Programa desenvolvido para calcular a média de notas com entrada de dados pelo usuário.

---

## 🚀 Sobre o projeto

Este projeto foi criado com o objetivo de praticar conceitos fundamentais da linguagem C, incluindo entrada de dados, operações matemáticas e lógica de programação.

O programa solicita ao usuário três notas, realiza o cálculo da média e exibe o resultado no terminal.

---

## 📌 Conceitos aplicados

✔ Entrada e saida de dados 
✔ Uso de variáveis do tipo `float`  
✔ Cálculo de média aritmética  
✔ Conversão de tipos (casting)  

---

## ⚙️ Funcionamento

O programa executa os seguintes passos:

1. Solicita três notas ao usuário
2. Armazena os valores em variáveis
3. Calcula a média das notas
4. Exibe o resultado formatado

---

## 🖥️ Exemplo de uso

```bash
Digite a primeira nota: 7.5
Digite a segunda nota: 8.0
Digite a terceira nota: 6.5

Média: 7.33
```
